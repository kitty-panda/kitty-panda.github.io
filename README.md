### Linux Commands
  1. List Directory Contents
```bash
    ls
```
or
```bash
    ls -l
```
or
```bash
    ls -alsh
```
  2. Print Name Of Current/Working Directory
```bash
    pwd
```
  3. Change Directory
```bash
    cd <path-of-directory>
```
