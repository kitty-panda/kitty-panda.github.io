# Linux Commands
1. List Directory Contents
    ```bash
    ls
    ```
      or
    ```bash
    ls -l
    ```
    or
    ```bash
    ls -alsh
    ```
1. Print Name Of Current/Working Directory
    ```bash
    pwd
    ```
1. Change Directory
    ```bash
    cd <path-of-directory>
    ```
